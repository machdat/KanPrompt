>_ KanPrompt

Öffne den Projektordner (z.B. erju_wp27/).
KanPrompt sucht darin doc/prompts/ mit backlog-priority.json und den Unterordnern new/, in-progress/, done/.
Falls die Struktur noch nicht existiert, wird sie automatisch angelegt.

Drag & Drop verschiebt Dateien zwischen Ordnern und aktualisiert die JSON automatisch.

Benötigt Chrome oder Edge · File System Access API
