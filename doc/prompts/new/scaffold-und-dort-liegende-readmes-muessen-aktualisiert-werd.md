# Scaffold und dort liegende readmes müssen aktualisiert werden. Ist Scaffold überhaupt Teil von KanPrompt?

## Problem / Motivation

Die Funktionalität von KanPrompt ist unter anderem in der SCaffold Struktur liegend. Es ist zu prüfen ob die Scaffold Struktur schon auf github committed ist. Ausserdem muss ein update script geschreben werden, das die Änderungen auf andere Projekte übertragen und initialisieren kann. Dieses Script muss automatisch für jede Änderung der Version von KanPrompt ausgeführt werden (genau 1x). Das Repository muss wissen zu welcher KanPrompt-Version es gehört. Entsprechend muss dann das Skript die noch fehlenden Änderungen vornehmen.
